"Nowadays, programming is a very crucial skill. In this course, we'll learn some topics of advanced programming, including but not limited to object-oriented programming, data structures and algorithm analysis. Although all of these topics are independent of a special programming language, we use one the most liked programming languages, called {Python}! We chose Python because it's simple and open source. In this course, our main focus is on your teamwork and programming activities. You'll learn Object-Oriented programming, and you'll have to implement a project as a final teamwork result. You'll be expected to write classes, use data structures and design a graphical user interface for your project. For more information visit the {Syllabus} section below."

Some Important Notes:

  1. You should use Object-Oriented programming in your project and the application program interfaces (API) will be demanded.
  2. You're not limited to the taught topics for your project, you can use whatever you want as long as you know it well.
  3. Prepare an explanation file for your project (no need to explain every line though).
  4. You have to design a graphical user interface (GUI) for your project.
  5. Copied projects are easily detected and they result in a 0.
